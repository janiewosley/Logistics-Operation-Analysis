## 📖 Project Overview

The Logistics Operations Analysis Dashboard is an end-to-end Business Intelligence solution built in Power BI to provide operational visibility into logistics performance.

The report consist of financial, operational, fleet, driver, maintenance, fuel, customer, and safety data into an interactive dashboard, enabling decision-makers to monitor performance, identify trends, and improve operational efficiency.

---

# Business Problem

Logistics organizations generate large volumes of operational data every day. Without centralized reporting, it becomes difficult to answer important business questions such as:

- Which cities generate the highest revenue?
- Which drivers perform best?
- Are maintenance costs increasing?
- Which trucks are underutilized?
- How much fuel is being consumed?
- Are safety incidents increasing?
- Which customers generate the most revenue?

This project addresses those challenges by transforming raw logistics data into an interactive analytics solution.
